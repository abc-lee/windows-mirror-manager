# Windows全局镜像管理器 - UI设计系统

基于ui-ux-pro-max技能生成的设计系统文档

## 项目背景
Windows全局镜像管理器 - 单EXE应用，管理Git、Pip、HuggingFace镜像源

## 设计系统概览

### 设计模式：Interactive Demo + Feature-Rich
- **CTA位置**：界面顶部
- **主要区域**：功能区 > 状态显示 > 操作区

### 风格：Micro-interactions（微交互）
- **关键词**：小型动画、基于手势、触觉反馈、微妙动画、上下文交互、响应式
- **最适合**：移动应用、触屏UI、生产力工具、用户友好、消费者应用、交互组件
- **性能**：⚡ 优秀 | **无障碍**：✓ 良好

### 组件规范

#### 1. 下拉框 (CTkComboBox)
- **背景色**：白色 #FFFFFF
- **边框色**：#E0E0E0
- **悬停色**：#F5F5F5
- **字体**：Inter，12pt
- **选项**：每个下拉框第一个选项为"原始/官方"

#### 2. 测试按钮 (CTkButton)
- **尺寸**：60x30像素
- **背景色**：主色 #0D9488
- **悬停色**：#0B8378
- **文字**："测试"
- **动画**：点击时显示旋转图标，完成后显示延时时间

#### 3. 应用按钮 (CTkButton)
- **尺寸**：120x40像素
- **背景色**：行动色 #F97316
- **悬停色**：#E06315
- **文字**："应用"
- **位置**：底部居中

#### 4. 状态标签 (CTkLabel)
- **正常状态**：深青色 #134E4A
- **进行中状态**：辅助色 #14B8A6
- **成功状态**：绿色 #10B981
- **错误状态**：红色 #EF4444
- **字体**：Inter，11pt，斜体

### 动画规范
1. **测试按钮旋转动画**：使用CTkProgressBar模拟旋转
2. **状态切换淡入淡出**：透明度从0到1，时长200ms
3. **应用进度显示**：进度条动画

### 交互反馈
1. **即时反馈**：所有操作都有状态更新
2. **进度可见**：长时间操作显示进度条
3. **错误处理**：明确错误提示和解决方案
4. **权限提示**：需要管理员权限时清晰提示

### 实现要点
1. 使用CustomTkinter 5.2.2
2. 适配Windows 11视觉风格
3. 确保浅色主题一致性
4. 实现所有微交互效果
